# About virtualsoils.ca
virtualsoils.ca is a project started by Amy Wells at the University of British Columbia with the goal of making high quality, 3D models of soils freely available to students, and science communicators in a global scale.

Following some initial testing, and the securing of a UBC Teaching and Learning Enhancement Fund, the UBC Emerging Media Lab became collaborators in developing the tool to have features critically important for educational use.
https://eml.ubc.ca/projects/virtual-soils/
